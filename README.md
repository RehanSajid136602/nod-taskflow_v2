# TaskFlow - Minimalist Task Manager

A clean, modern, and high-performance task management web application built with Vanilla JavaScript.

## Features

- **Task Management**: Create, edit, delete, and complete tasks with ease.
- **Organization**: Assign priority levels (High/Medium/Low) and categories.
- **Productivity Dashboard**: Real-time statistics of your tasks.
- **Smart Filtering**: Filter by status (Active/Completed) and category.
- **Sorting**: Order tasks by date or priority.
- **Search**: Instantly find tasks with the search bar.
- **Drag & Drop**: Reorder your tasks naturally.
- **Persistence**: Your data is saved automatically to your browser's local storage.
- **Keyboard Shortcuts**:
  - `N`: New Task
  - `Esc`: Close Modal / Cancel

## Technical Details

- **Frontend**: HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript (ES6+).
- **Design**: Mobile-first, responsive, minimalist aesthetic using the Inter font.
- **State Management**: Custom JavaScript class-based state handling.

## How to Run

Simply open `index.html` in any modern web browser.
